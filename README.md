# Aether Lens — Frontend Client

A high-performance web interface built with **Next.js (App Router)** and **Tailwind CSS** designed for tracking, managing, and evaluating global supply chain operational hubs and risk metrics.

## 🚀 Key Features

* **Global Risk Index Dashboard:** Explore dynamic supplier profiles with live search, category filtering, risk-tier segregation, and fully interactive **pagination controls**.
* **AI-Assisted Classification Node:** Real-time integration with backend AI systems to auto-classify facility logs and generate tracking tags on the fly.
* **Secured User Authentication:** Integrated with Better-Auth for seamless user identification and scoped record management.
* **Responsive Card Layout:** Skeleton loading UI state handlers and modern micro-interactions for optimal user experience.
**Autonomous Hero Intelligence Hub:** Interactive landing page featuring dynamic rotating taglines, glowing cryptographic background grids, and direct action triggers.
* **AI Agent Telemetry Dashboards:** Direct communication interfaces with the backend Gemini-powered risk engine for automated supplier log analysis, risk matrix reporting, and auto-classification tag generation.
* **Enterprise Footer Infrastructure:** Includes a functional real-time telemetry newsletter subscription box, live node synchronization status indicator, and operational directory navigation links.

---

## 🛠️ Tech Stack

* **Framework:** Next.js (React, App Router, TypeScript)
* **Styling:** Tailwind CSS
* **Notifications:** React Toastify
* **HTTP Client:** Native Fetch API
